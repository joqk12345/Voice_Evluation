# 图片资源说明

本目录用于存放小程序的图片资源文件。

## 需要的图片文件

### 图标文件
- `home.png` - 首页图标
- `home-active.png` - 首页激活图标
- `record.png` - 录音图标
- `record-active.png` - 录音激活图标
- `history.png` - 历史图标
- `history-active.png` - 历史激活图标
- `profile.png` - 个人中心图标
- `profile-active.png` - 个人中心激活图标

### 默认图片
- `default-avatar.png` - 默认头像
- `share-cover.png` - 分享封面图
- `share-result.png` - 结果分享图
- `share-record.png` - 记录分享图
- `share-history.png` - 历史分享图
- `share-recommend.png` - 推荐分享图
- `share-profile.png` - 个人中心分享图

### 二维码图片
- `qrcode.png` - 小程序二维码图片（用于分享图片中显示）
  - 建议尺寸: 500x500px 或更大（正方形）
  - 格式: PNG（支持透明背景）
  - 内容: 包含Super Singer品牌特色的二维码，中心有卡通麦克风角色
  - 注意: 如果图片不存在，系统会自动使用模拟二维码作为降级方案

## 图片规格建议

### 图标规格
- 尺寸: 81x81px
- 格式: PNG
- 背景: 透明

### 分享图片规格
- 尺寸: 500x400px
- 格式: PNG/JPG
- 背景: 白色或透明

### 头像规格
- 尺寸: 200x200px
- 格式: PNG/JPG
- 背景: 透明或白色

## 注意事项

1. 所有图片文件需要放在此目录下
2. 文件名需要与代码中的引用保持一致
3. 建议使用PNG格式以获得更好的显示效果
4. 图片大小建议控制在100KB以内以优化加载速度
