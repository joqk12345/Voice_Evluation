# 快速启动指南

## 🚀 快速开始

### 1. 环境准备
- 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号并获取 AppID

### 2. 项目导入
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目目录：`Voice_Evluation-V1`
4. 输入你的 AppID（或选择测试号）
5. 点击"导入"

### 3. 配置项目
1. 修改 `project.config.json` 中的 `appid` 为你的小程序 AppID
2. 在 `app.js` 中配置你的后端 API 地址（如果需要）

### 4. 预览运行
1. 点击工具栏的"预览"按钮
2. 使用微信扫码在手机上预览
3. 或点击"真机调试"进行调试

## 📱 功能测试

### 基础功能测试
1. **登录功能**
   - 点击"微信一键登录"
   - 授权用户信息
   - 验证登录状态

2. **录音评测**
   - 进入录音页面
   - 选择评测时长
   - 开始录音并测试录音功能
   - 查看评测结果

3. **历史记录**
   - 查看评测历史
   - 测试记录删除功能
   - 验证统计数据

4. **个性化推荐**
   - 查看练习计划推荐
   - 浏览推荐歌曲
   - 测试每日挑战

### 界面测试
- 检查所有页面的布局和样式
- 验证按钮点击效果
- 测试页面跳转功能
- 确认分享功能

## 🔧 开发调试

### 调试工具
- 使用微信开发者工具的调试面板
- 查看 Console 日志输出
- 使用 Network 面板检查网络请求
- 使用 Storage 面板查看本地存储

### 常见问题
1. **录音权限问题**
   - 确保在真机上测试录音功能
   - 检查录音权限设置

2. **页面跳转问题**
   - 检查页面路径是否正确
   - 确认页面已在 app.json 中注册

3. **样式显示问题**
   - 检查 WXSS 文件语法
   - 确认样式类名正确

## 📦 发布部署

### 1. 代码上传
1. 点击工具栏的"上传"按钮
2. 填写版本号和项目备注
3. 点击"上传"

### 2. 提交审核
1. 登录微信公众平台
2. 进入小程序管理后台
3. 提交审核并等待审核结果

### 3. 发布上线
1. 审核通过后点击"发布"
2. 小程序正式上线

## 🎨 自定义配置

### 修改配色方案
在 `app.wxss` 中修改 CSS 变量：
```css
page {
  --primary-color: #008080; /* 主色调 */
  --secondary-color: #DAA520; /* 辅助色 */
  --text-dark: #333333; /* 深色文字 */
  --text-light: #666666; /* 浅色文字 */
  --background-light: #f5f5f5; /* 背景色 */
  --white: #ffffff; /* 白色 */
}
```

### 修改页面配置
在 `app.json` 中修改页面配置：
```json
{
  "pages": [
    "pages/index/index",
    // 添加或删除页面
  ],
  "tabBar": {
    // 修改底部导航栏
  }
}
```

### 添加新功能
1. 在 `pages` 目录下创建新页面
2. 在 `app.json` 中注册新页面
3. 实现页面逻辑和样式
4. 添加页面间的跳转逻辑
