# 声乐评测微信小程序

## 项目简介

声乐评测微信小程序是一个专业的声乐评测工具，帮助用户进行自我评估与改进。通过录音分析用户的音准、节奏、音量、音色等多方面的表现，提供个性化的评测结果和练习建议。

## 功能特性

### 🎤 核心功能
- **声乐评测**: 支持30秒到2分钟的录音评测
- **多维度分析**: 音准、节奏、音量、音色四个维度评分
- **实时录音**: 可视化录音界面，实时显示录音状态
- **历史记录**: 保存评测历史，支持对比分析

### 🎵 个性化推荐
- **练习计划**: 根据评测结果生成个性化练习方案
- **歌曲推荐**: 推荐适合用户水平的歌曲
- **技巧指导**: 提供声乐技巧教学和视频指导
- **每日挑战**: 设置每日练习挑战任务

### 📊 数据分析
- **详细报告**: 提供详细的评测报告和改进建议
- **趋势分析**: 显示用户进步趋势图表
- **成就系统**: 记录用户成就和里程碑
- **统计概览**: 总评测次数、最高分、平均分等统计

### 🔐 用户系统
- **微信登录**: 支持微信一键登录
- **个人资料**: 用户信息管理和等级系统
- **数据同步**: 评测数据云端同步保存

### 📱 社交分享
- **结果分享**: 分享评测结果到微信朋友圈
- **H5页面分享**: 支持生成H5页面链接，可在微信、朋友圈、其他平台分享
- **分享图片生成**: 自动生成精美的分享图片，包含评测结果和二维码
- **成就分享**: 分享成就徽章和进步记录
- **邀请好友**: 邀请朋友一起使用评测功能

## 技术架构

### 前端技术
- **框架**: 微信小程序原生开发
- **语言**: JavaScript, WXML, WXSS
- **UI设计**: 采用青绿色和金色配色方案
- **交互**: 流畅的动画效果和用户反馈

### 后端技术
- **API**: RESTful API设计
- **数据库**: 支持MySQL/MongoDB
- **音频处理**: 音频信号处理和机器学习算法
- **云服务**: 支持微信云开发

### 音频分析算法
- **音准分析**: 基于频谱分析检测音高准确性
- **节奏分析**: 检测音频的节奏与拍子稳定性
- **音量分析**: 根据音频信号强度评估音量控制
- **音色分析**: 基于音频特征分析声音质量

## 安装使用

### 环境要求
- 微信开发者工具
- Node.js (可选，用于后端开发)
- 微信小程序账号

### 开发步骤
1. 下载微信开发者工具
2. 导入项目到开发者工具
3. 配置小程序AppID
4. 预览和调试
5. 上传代码并发布

### 配置说明
1. 修改 `app.json` 中的小程序配置
2. 更新 `utils/util.js` 中的API接口地址
3. 配置音频分析服务接口
4. 设置用户数据存储
5. **H5分享配置**（可选）：
   - 部署 `h5/result.html` 到服务器
   - 在 `app.js` 中配置 `h5BaseUrl`
   - 详细配置请参考 [H5_SHARE_SETUP.md](./H5_SHARE_SETUP.md)
6. **二维码配置**（可选）：
   - 将二维码图片放置到 `images/qrcode.png`
   - 详细说明请参考 [images/QRCODE_SETUP.md](./images/QRCODE_SETUP.md)
